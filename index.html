gesture What is this about?

 

Developer Student Club BVP in collaboration with Google Developers and Qwiklabs brings to you the #GCPCrashCourse, the best place to get started with the Google Cloud Platform. Do sign up in order to have a seamless experience for #GCPCrashCourse.

 

Fill this first before you start anything.

 

 

 

How do we do?

We have 3 Exciting Tracks for you. Click on a Track to get enrolled in it.

You need to complete ALL the listed Quests of EACH track.

 

Application Development

Finish ALL of the following Set of Quests to complete this Track
  • Baseline : Deploy and Develop
  • Websites & Web Applications
  • Deploying Applications
  • Cloud Development
  • Exploring APIs on Google Cloud Platform

 

Data Science with GCP   

Finish ALL of the following Set of Quests to complete this Track
  • Baseline : Data, ML & AI
  • BigQuery for Data Analysis
  • Scientific Data Processing
  • Data Engineering
  • Data Science on Google Cloud Platform

 

Machine Learning with GCP

Finish ALL of the following Set of Quests to complete this Track
  • Baseline : Data, ML & AI
  • Intro to ML : Language Processing / Image Processing
  • BigQuery for ML
  • Tensorflow on GCP
  • Machine Learning APIs
